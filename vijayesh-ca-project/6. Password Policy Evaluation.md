## 6. Recommended Password Policy

### 6.1 Immediate Changes (Week 1-2)

```markdown
## PASSWORD POLICY v2.0

### Minimum Requirements:
✓ Minimum length: 12 characters (STRONGLY RECOMMEND 16+)
✓ Maximum length: 128 characters
✓ Allow all printable ASCII characters
✓ Allow emoji and spaces

### Prohibited:
✗ Common passwords (top 10,000)
✗ Personal information (name, DOB, username)
✗ Sequential patterns (123, abc, qwerty)
✗ Repeating characters (aaa, 111)

### Hashing Requirements:
✓ Algorithm: Argon2id (RECOMMENDED) or bcrypt (ACCEPTABLE)
✓ Salt: Unique 16+ byte salt per password
✓ Work factor: Minimum 12 for bcrypt
✓ Memory: 64MB minimum for Argon2
```

### 6.2 Technical Implementation

```python
# ✅ RECOMMENDED - Argon2id Implementation
from argon2 import PasswordHasher

ph = PasswordHasher(
    time_cost=3,
    memory_cost=65536,
    parallelism=4,
    hash_len=32,
    salt_len=16
)

# Hash password
hash = ph.hash(user_password)

# Verify password
try:
    ph.verify(hash, user_password)
except VerifyMismatchError:
    return "Invalid password"
```

```python
# ✅ ACCEPTABLE - bcrypt Implementation
import bcrypt

# Hash password (work factor 12)
salt = bcrypt.gensalt(rounds=12)
hashed = bcrypt.hashpw(password.encode(), salt)

# Verify
if bcrypt.checkpw(password.encode(), hashed):
    print("Password valid")
```
